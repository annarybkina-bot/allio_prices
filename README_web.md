# Веб-приложение для анализа квартир

## Запуск приложения

1. Установите необходимые зависимости:
```bash
pip install flask matplotlib numpy
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте в браузере:
```
http://localhost:5001
```

## Использование

1. **Загрузка файлов:**
   - Загрузите файл основного ЖК (обязательно)
   - Загрузите файлы конкурентов (можно несколько)

2. **Создание групп:**
   - Нажмите кнопку "Создать группы"
   - Будет показан список всех созданных групп

3. **Сравнение групп:**
   - Нажмите кнопку "Сравнить сопоставимые группы"
   - Для каждой пары сопоставимых групп будет показано:
     - Boxplot с двумя графиками
     - Гистограмма с двумя графиками
     - Статистика: макс, мин, медиана, среднее, выбросы
