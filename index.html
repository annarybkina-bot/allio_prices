Цены конкурентов. Сравнение и анализ

Веб-приложение для анализа и группировки квартир по сопоставимым параметрам.

Этот проект — Flask-приложение. На GitHub Pages запускается только статическая страница. Чтобы запустить приложение локально:
  1. Клонируйте репозиторий: git clone https://github.com/annarybkina-bot/allio_prices.git
  2. Установите зависимости: pip install flask matplotlib numpy plotly
  3. Запустите: python app.py
  4. Откройте в браузере: http://localhost:5001
